# GFE Challenges

GFE Challenges is a collection of frontend projects created by Mark Austria to practice and showcase web development skills.

## About

This project includes a set of UI challenges built with modern frontend tools and patterns. Each challenge focuses on clean structure, reusable components, and responsive design.

## Purpose

- Practice frontend implementation.
- Build reusable UI components.
- Showcase design and development skills.
- Track progress across multiple challenge pages.

## Current Challenge

- Testimonial Card

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Next/Image
- Google Fonts

## Structure

- `app/page.tsx` for the home page.
- `app/testimonial-card/page.tsx` for the testimonial card demo.
- `components/TestimonialCard.tsx` for the reusable card component.

## Notes

The project is designed to grow over time as more challenges are added.
